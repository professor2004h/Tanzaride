Name the ZIP: tanzaride.zip


STEP 2: LOGIN TO HOSTINGER
---------------------------
1. Go to: https://hpanel.hostinger.com
2. Enter your email and password
3. Click on your domain (tanzaride.com)


STEP 3: OPEN FILE MANAGER
--------------------------
1. In left sidebar, click "Files"
2. Click "File Manager"
3. You'll see "public_html" folder
4. Click to open it


STEP 4: CLEAN public_html
--------------------------
⚠️ IMPORTANT: Delete any existing files in public_html

1. Select all files (if any exist)
2. Right-click → Delete
3. Confirm deletion


STEP 5: UPLOAD ZIP
------------------
1. Click "Upload Files" button (top right)
2. Select your tanzaride.zip file
3. Wait for upload to complete (progress bar)


STEP 6: EXTRACT ZIP
-------------------
1. Find tanzaride.zip in the file list
2. Right-click on it
3. Select "Extract"
4. Destination: /public_html
5. Click "Extract" button
6. Wait for extraction to complete


STEP 7: DELETE ZIP FILE
------------------------
1. Right-click on tanzaride.zip
2. Select "Delete"
3. Confirm

STEP 9: ENABLE SSL (HTTPS)
---------------------------
1. In hPanel, go to "Security" → "SSL"
2. Find your domain
3. If SSL not installed, click "Install SSL" (FREE!)
4. Toggle "Force HTTPS" to ON
5. Wait 2-3 minutes

🔧 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PROBLEM: Website shows "Index of /" or directory listing
SOLUTION: Make sure index.html is directly in public_html (not in a subfolder)

PROBLEM: Images not loading
SOLUTION: 
  1. Check assets/ folder is in public_html
  2. Verify image file names match exactly (case-sensitive)
  3. Clear browser cache (Ctrl+F5)

PROBLEM: CSS not working (website looks plain)
SOLUTION: 
  1. Verify style.css is in public_html root
  2. Clear browser cache (Ctrl+F5)

PROBLEM: HTTP not redirecting to HTTPS
SOLUTION:
  1. Check .htaccess file is uploaded
  2. Enable "Force HTTPS" in hPanel → Security → SSL
  3. Wait 5 minutes, then test

PROBLEM: 404 errors on pages
SOLUTION:
  1. Verify all .html files are in public_html root
  2. Check file names are correct (case-sensitive)

PROBLEM: Gallery/Filters not working
SOLUTION:
  1. Verify all .js files are uploaded
  2. Press F12, check Console for errors
  3. Clear browser cache
